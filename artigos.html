As tecnologias da Web

HTML

O Html (Linguagem de marcação de hipertexto) é a linguagem padrão essencial para criar uma página Web. Ele é o esqueleto de um site, que depois é estilizado com CSS e tornado interativo com Javascript. Ela estrutura o conteúdo define os títulos, parágrafos, listas, rodapés , imagens, organizando a informação logicamente que será interpretada pelo navegador e renderizada para o usuário. Ele também cria os hiperlinks conectando diferentes páginas e sites, oque é a base da Web.

CSS

O CSS (Folhas de Estilo em Cascata) é a linguagem para definir o estilo, layout, e a aparência visual de páginas Web em HTML. Ele controla cores, fontes, tamanhos, espaçamentos e posicionamentos de elementos. Com o CSS é possível tornar sites responsivos, adaptando-os a diferentes dispositivos e tamanhos de tela.

Javascript

O javascript é a linguagem de programação que está presente em mais de 90% da web, interpretada e versátil, essencial para o desenvolvimento web moderno. Ele torna sites interativos e dinâmicos, manipulando o conteúdo (DOM) diretamente no navegador, permitindo funcionalidades como atualizações em tempo real, animações, mapas e validação de formulários sem carregar a página. No Desenvolvimento Backend (Servidor) com o NodeJs, o javascript pode ser usado no servidor para gerenciar banco de dados, APis e lógicas complexas de backend. O javascript forma, junto com o HTML e CSS, a base da tecnologia web moderna.
